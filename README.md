# Snappics

### *Strike a pose, Capture the Joy, Snap, Share, and Repeat.*

> A fun, browser-based photobooth experience — no app downloads, no sign-ups, just pure nostalgic vibes right in your browser.

---

## What is Snappics?

**Snappics** is a lighthearted photobooth web app built just for fun! Inspired by those classic sticker photo booths you'd find at malls and arcades, Snappics brings the experience straight to your browser. Snap a four-photo sequence, dress it up with frames and stickers, and walk away with a downloadable photo strip to keep or share.

No backend. No fuss. Just smiles.

---

## Features

- **Four-Photo Capture Sequence** — Automatically captures a series of four photos, just like a real photobooth
- **Customizable Frames** — Choose from a variety of fun frames to style your photo strip
- **Sticker Overlays** — Add playful stickers to personalize each shot
- **Downloadable Photo Strip** — Save your finished strip as an image to keep, print, or share
- **Fully Browser-Based** — Built with plain HTML and Tailwind CSS — no frameworks, no installs, just open and go

---

## 🛠️ Built With

- **Tailwind CSS** — For quick, clean, utility-first styling

## Getting Started

Since Snappics is plain HTML + CSS, getting it running is as simple as it gets:

```bash
# Clone the repo
git clone https://github.com/your-username/snappics.git

# Open in your browser
open index.html
```

Or just drag `index.html` into your browser. That's it!
